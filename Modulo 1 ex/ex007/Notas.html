cuidado quanto as diferenças do html 4 para o 5, diversas tags ficaram obsoletas por conta da mudança

a principal diferença é que o Html se tornou uma linguagem que foca masi na semantica do que na forma das tags
isso significa que ela foca mais nos significados das palavras ao inves do estilo

o estilo agora é feito praticamente todo em CSS

HTML esta focado agora na semantica, no significado, mudar cor, tamanho mudar o estilo da página, fica reservado para o CSS

mesmo que o VS não marque como errado, ficar sempre atento nas tags que estão obsoletas e usar as linguagens somente para aquilo que elas são designadas

HTML
para escrita, o bruto da página
CSS
para das estilo as páginas, cores fontes e tudo mais
JavaScript
para fazer animação na página
