Principais formatações

Negrito / Destaque

Existem duas maneiradas de colocar Negrito, uma maneira semantica e uma não semantica

Nessa fraze tem ma termo em negrito utilizando a tag <b> (não semantica)

Nesta fraze tem um termo em Destaque utilizando a tag <strong> (forma semantica)

Nesta fraze temos um termo em italico utilizando a tag <i> (não semantica)

Nesta fraze temos um termo em ênfase utilizando a tag <em> (forma semantica)

Recomendação da W3C: é forma CSS é sentido HTML


Texto Marcado

Podemos criar também um texto marcado utilizando a tag <mark>(forma semantica)

Obs: para mudar a cor do marcador é necesario utilizar CSS, o exemplo usado nesta página o style é colocado dentro da head e isso faz com que todos os mark da pagina fiquem dessa cor, porem se colocado dentro do mark texto marcado, como feito no exemplo anterior ele mudara a forma somente deste.

Testo grande e pequeno

Estamos criando um Texto Grande e um texto pequeno nesse parágrafo utilizando as tags <big> para a grande e a <small> para a pequena
Obs: a tag <big> pode ter seu uso desatifado por ser considerado uma tag obsoleta, ficar atento quanto a isso, para deixar a letra grande utilizar o CSS

Texto deletato

Podemos marcar um texto como deletado para indicar que ele deve ser lido mas não considerado utilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois utilizando a tag <ins>

existe também o sublinhado utilizando a tag <u> porem ela pode cair em desuso (não semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 utiliza-se a tag <sup> na parte desejada

texto subscrito

Para inserir coisas do tipo H2O utiliza-se a tag <sub> na parte desejada
